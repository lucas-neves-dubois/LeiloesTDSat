# Sistema LeilõesTDSat

## Descrição

O **Sistema LeilõesTDSat** é uma aplicação para gerenciar o cadastro de produtos e seus valores, facilitando a realização de leilões. Permite cadastrar produtos, listá-los em uma tabela e gerenciar vendas em leilão.

## Tecnologias Utilizadas

- **Java**
- **MySQL**
